# Good Old Games

This is application called **Good Old Games**. In this application you will be able to play different games like:

- Rock Paper Scissors
- Pig Game
- Hangman
- Guess a Number

The application is made in **React** and Package.json is also provided.

## Libraries Used In Project

- React Router
- Uuid
